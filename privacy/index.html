Политика конфиденциальности

Настоящая политика описывает, какие данные могут обрабатываться при использовании сайта, Discord-бота и сервисов проекта ELYSIUM DAYZ.

1. Какие данные обрабатываются

  • Discord ID, имя пользователя и публичные данные Discord-аккаунта.
  • Информация о заказах, суммах платежей и статусах начисления.
  • Технические данные: IP-адрес, user-agent, дата и время обращения к сайту.
  • Email или иные контакты, если пользователь обращается в поддержку.

2. Для чего используются данные

  • Для создания и обработки заказов на пополнение внутреннего баланса.
  • Для автоматического начисления баланса после оплаты.
  • Для поддержки пользователей и разбора спорных ситуаций.
  • Для безопасности, предотвращения повторного начисления и защиты от мошенничества.

3. Передача данных платёжной системе

При оплате пользователь переходит на страницу платёжной системы. Обработка платёжных данных банковской карты или другого способа оплаты выполняется платёжной системой, а проект получает только техническое подтверждение статуса оплаты, номер заказа, сумму и служебные параметры.

4. Срок хранения

Данные о заказах и начислениях могут храниться в течение срока, необходимого для исполнения услуги, бухгалтерского и технического учёта, рассмотрения обращений и соблюдения требований закона.

5. Контакты по вопросам данных

По вопросам обработки данных можно обратиться по адресу: support@elysium-dayz.ru.
